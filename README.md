# NEDO.DEV

Сайт студии NEDO.DEV. Черный фон, шум, странные решения и пасхалки — все как мы любим.

## Стек

- Vite + React + TypeScript
- Tailwind через CDN

## Запуск

1. `npm install`
2. `npm run dev` (http://localhost:8000)

## Скрипты

- `npm run dev` — локальная разработка
- `npm run build` — прод сборка
- `npm run preview` — предпросмотр сборки
- `npm start` — alias для dev

## Деплой

Собранные файлы лежат в `dist/`.
